# 🚀 Enterprise RAG AI Chatbot

> A Professional AI-Powered PDF Chatbot built using Google Gemini, Streamlit and a Lightweight Local Vector Store.

---

## 🌟 Overview

Enterprise RAG AI Chatbot allows users to upload PDF documents and ask questions directly from their documents.

The system extracts text from PDFs, creates embeddings, stores them inside a lightweight local vector store, retrieves the most relevant context, and generates accurate answers using Google Gemini.
